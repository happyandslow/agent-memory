# agent-memory Skill — Phase 1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build and install the `agent-memory` skill — one shared `PLAYBOOK.md` procedure plus deterministic Python helpers — invoked automatically by Claude Code (Stop hook) and Hermes (daily cron), on both Mac and remote machines.

**Architecture:** A self-contained skill package lives in the `claude-skills` repo. `PLAYBOOK.md` is the single-source procedure; a Claude `SKILL.md` and a slimmed Hermes skill both delegate to it. Deterministic, testable Python scripts (`memory_lib.py`, `check_memory_repo.py`, `build_views.py`, `find_conflicts.py`) do the mechanical work and take `--root <AGENT_MEMORY_ROOT>` so the same code runs everywhere. `install.sh` clones the repo and symlinks the skill into each harness. Phase 1 builds/installs only; Phase 2 (separate plan) runs the skill to migrate existing projects.

**Tech Stack:** Python 3.11+ (stdlib only — `pathlib`, `re`, `argparse`, `datetime`), `pytest` for tests, Bash for `install.sh`, Markdown for the skill docs.

## Global Constraints

- Python scripts use **stdlib only** (no third-party deps); target Python 3.11+.
- Every script accepts `--root <path>` resolving the agent-memory repo; resolution order: explicit `--root`, then `$AGENT_MEMORY_ROOT`, then sibling `../agent-memory`, then Mac fallback `/Users/lexu/Projects/agent-memory`.
- Generated view files carry the exact header line: `<!-- GENERATED by agent-memory skill: do not hand-edit -->`
- Dated-file prefix regex is exactly `^\d{4}-\d{2}-\d{2}-`.
- Project discovery skips `_template`, hidden dirs (name starting with `.`), and non-directories under `projects/`.
- Scripts must be deterministic and side-effect-free unless invoked with a write flag; tests never depend on wall-clock time (inject via `--now`).
- Skill package path in the skills repo: `claude-skills/agent-memory/`.
- The Claude Code skill name is exactly `agent-memory`.
- Never auto-resolve ambiguous human conflicts; scripts only *detect and report* them.

```python
# memory_lib.py
from __future__ import annotations
import os
import re
from pathlib import Path

DATE_PREFIX_RE = re.compile(r"^\d{4}-\d{2}-\d{2}-")
GENERATED_HEADER = "<!-- GENERATED by agent-memory skill: do not hand-edit -->"
MAC_FALLBACK = Path("/Users/lexu/Projects/agent-memory")

def resolve_root(arg: str | None) -> Path:
    if arg:
        return Path(arg).expanduser().resolve()
    env = os.environ.get("AGENT_MEMORY_ROOT")
    if env:
        return Path(env).expanduser().resolve()
    sibling = Path.cwd().parent / "agent-memory"
    if (sibling / "projects").is_dir():
        return sibling.resolve()
    return MAC_FALLBACK

def discover_projects(root: Path) -> list[Path]:
    projects = root / "projects"
    if not projects.is_dir():
        return []
    return sorted(
        p for p in projects.iterdir()
        if p.is_dir() and not p.name.startswith(".") and p.name != "_template"
    )

def iter_dated_files(project: Path) -> list[tuple[str, Path]]:
    out: list[tuple[str, Path]] = []
    for path in project.rglob("*"):
        if ".git" in path.parts or not path.is_file():
            continue
        if DATE_PREFIX_RE.match(path.name):
            out.append((path.name[:10], path))
    return out
```

```python
# check_memory_repo.py
from __future__ import annotations
import argparse
from pathlib import Path
from memory_lib import resolve_root, discover_projects, DATE_PREFIX_RE

FORBIDDEN_SUFFIXES = {".db", ".sqlite", ".sqlite3", ".pem", ".key", ".p12"}
FORBIDDEN_NAMES = {".env", "id_rsa", "id_ed25519"}

def check(root: Path) -> list[str]:
    problems: list[str] = []
    for project in discover_projects(root):
        for path in project.rglob("*"):
            if ".git" in path.parts or not path.is_file():
                continue
            name = path.name.lower()
            rel = path.relative_to(root)
            if name in FORBIDDEN_NAMES or any(name.endswith(s) for s in FORBIDDEN_SUFFIXES):
                problems.append(f"potential secret/db file: {rel}")
            if "docs" in path.parts and not DATE_PREFIX_RE.match(path.name):
                if not path.name.startswith("."):
                    problems.append(f"undated docs artifact: {rel}")
    return problems

def main() -> int:
    ap = argparse.ArgumentParser()
    ap.add_argument("--root")
    args = ap.parse_args()
    root = resolve_root(args.root)
    problems = check(root)
    print(f"agent-memory check root={root}")
    for p in problems:
        print(f"- {p}")
    print("FAIL" if problems else "PASS")
    return 1 if problems else 0

if __name__ == "__main__":
    raise SystemExit(main())
```

```python
# build_views.py
from __future__ import annotations
import argparse
from pathlib import Path
from memory_lib import (
    resolve_root, discover_projects, iter_dated_files, GENERATED_HEADER,
)

def build_timeline(project: Path) -> str:
    events = iter_dated_files(project)
    events.sort(key=lambda t: (t[0], t[1].name), reverse=True)
    lines = [GENERATED_HEADER, "", f"# {project.name} Timeline", ""]
    if not events:
        lines.append("_No dated events yet._")
    for date, path in events:
        rel = path.relative_to(project).as_posix()
        topic = path.name[11:].rsplit(".", 1)[0]
        lines.append(f"- {date} — {topic} → `{rel}`")
    return "\n".join(lines) + "\n"

def build_index(project: Path) -> str:
    lines = [
        GENERATED_HEADER, "", f"# {project.name}", "",
        "## Quick links", "",
        "- [[plan]] — roadmap/progress (human)",
        "- [[capture]] — braindrop (human)",
        "- [[memory/context]] — session-start packet (generated)",
        "- [[tracking/status]] — dashboard (generated)",
        "- [[memory/timeline]] — date index (generated)",
        "- [[memory/project]] — stable facts",
        "",
    ]
    topics = sorted((project / "memory" / "topics").glob("*.md"))
    if topics:
        lines.append("## Topics")
        lines.append("")
        for t in topics:
            if t.name != "README.md":
                lines.append(f"- [[memory/topics/{t.stem}]]")
        lines.append("")
    return "\n".join(lines) + "\n"

def write_views(project: Path) -> list[Path]:
    written: list[Path] = []
    tl = project / "memory" / "timeline.md"
    tl.write_text(build_timeline(project))
    written.append(tl)
    idx = project / "index.md"
    idx.write_text(build_index(project))
    written.append(idx)
    return written

def main() -> int:
    ap = argparse.ArgumentParser()
    ap.add_argument("--root")
    ap.add_argument("--project")
    args = ap.parse_args()
    root = resolve_root(args.root)
    projects = discover_projects(root)
    if args.project:
        projects = [p for p in projects if p.name == args.project]
    for project in projects:
        written = write_views(project)
        print(f"regenerated {project.name}: {[str(w.relative_to(root)) for w in written]}")
    return 0

if __name__ == "__main__":
    raise SystemExit(main())
```

```python
# find_conflicts.py
from __future__ import annotations
import argparse
from pathlib import Path
from memory_lib import resolve_root, discover_projects

VIEW_FILES = ("memory/context.md", "tracking/status.md", "index.md")
MIN_DUP_LEN = 40

def _empty_scaffold_dirs(project: Path) -> list[str]:
    out = []
    for d in project.rglob("*"):
        if ".git" in d.parts or not d.is_dir():
            continue
        entries = [e for e in d.iterdir() if e.name != ".DS_Store"]
        only_readme = entries and all(e.name == "README.md" for e in entries)
        if not entries or only_readme:
            out.append(f"empty scaffold dir: {d.relative_to(project).as_posix()}")
    return out

def _cross_file_dupes(project: Path) -> list[str]:
    seen: dict[str, set[str]] = {}
    for rel in VIEW_FILES:
        path = project / rel
        if not path.is_file():
            continue
        for line in path.read_text().splitlines():
            norm = line.strip().lstrip("#-* ").strip()
            if len(norm) >= MIN_DUP_LEN:
                seen.setdefault(norm, set()).add(rel)
    return [
        f"duplicated across {sorted(files)}: {norm[:60]}..."
        for norm, files in seen.items() if len(files) >= 2
    ]

def find(root: Path) -> dict[str, list[str]]:
    result: dict[str, list[str]] = {}
    for project in discover_projects(root):
        problems = _empty_scaffold_dirs(project) + _cross_file_dupes(project)
        if problems:
            result[project.name] = problems
    return result

def main() -> int:
    ap = argparse.ArgumentParser()
    ap.add_argument("--root")
    args = ap.parse_args()
    root = resolve_root(args.root)
    result = find(root)
    for slug, problems in result.items():
        print(f"[{slug}]")
        for p in problems:
            print(f"  - {p}")
    if not result:
        print("no conflicts/declutter items found")
    return 0

if __name__ == "__main__":
    raise SystemExit(main())
```

```markdown
# Write Contract — writing to agent-memory

Any agent (Claude Code, Hermes) writing durable memory follows these rules.

## Principle
During a live session, **capture to dated `inbox/` by default**. `topics/`
consolidation happens later in the maintain pass. New topic files come only
from curation — this keeps `topics/` few and clean.

## Placement
- **Default (session write):** `memory/inbox/YYYY-MM-DD-<topic>.md` using
  `templates/capture-entry.md`. Compact, high-signal — not a raw transcript.
- **Allowed exception:** append to an *existing* `memory/topics/<slug>.md`
  in place (dated `## Updates` subsection) when refining that topic's
  evergreen facts. **Never create new topic files** — that is the maintain
  pass's job.
- **Meeting slides:** `meetings/` — freeform names, co-edited. Pull before
  edit, one working file per deck, commit after; ask if unsure about unsaved
  human edits. Never hand-reorganize.
- **Generated views** (`context.md`, `tracking/status.md`, `memory/timeline.md`,
  `index.md`) are never hand-written; the skill produces them. They carry
  `<!-- GENERATED by agent-memory skill: do not hand-edit -->`.

## Rule of thumb
Evergreen → the topic note (existing only). Time-stamped → a dated inbox file.
Never a new bare file in topics/. Never hand-edit a generated view.
```
