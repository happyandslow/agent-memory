Decode, relay seam, and prefill on one wafer

Both pipelines are compiled into a single SdkLayout artifact and stacked vertically on the fabric: decode to the north, prefill to the south, with a task-less relay seam filling the gap. The whole scheme hinges on one shared column, PLACE_X, where both block regions begin.

Why PLACE_X is the whole trick

PLACE_X is the fabric X-column where block column 0 of the block region begins. Decode and prefill are built by two separate functions that each compute their west band differently — yet both arithmetic paths are engineered to land on the same value, HT_WIDTH_tail + 4. Because the two block regions occupy the identical set of columns (here 8–23), decode PE at column c sits directly above the prefill PE at column c — so the KV handoff is a straight vertical (north) shift, no lateral routing.

Two different west-band arithmetics, one shared PLACE_X. The run() assert HT_WIDTH_tail == P_BLOCK_SIZE/2 keeps the HT band exactly the right width so the block columns of both halves line up — the precondition for the seam to be a vertical shift.

Vertical stacking & the seam

Row placement is pure arithmetic in run(). Decode is pinned to the top; each region is stacked below the previous with the two-row seam in between.

  • The seam is only Pw wide (the block columns, x 8–23) — it does not span the west band. At the seam rows, the west band is busy with decode's mux (row 17) and prefill's demux (row 18); the seam and those regions are X-disjoint, so they coexist on the same rows.
  • build_relay() places a Pw × 2 code region at (PLACE_X, RELAY_Y). Its CSL body is empty — the host paints colors 17 and 21 with input=SOUTH, output=NORTH, turning the seam into pure prefill→decode transit wire. Color 22 stays reserved with no traffic.
  • The dashed guide line in the diagram marks one representative block column c: the blue decode PE and the amber prefill PE on it share the same fabric X, which is exactly why the KV shift travels straight up through the seam.
  • Footprint for this config: total_w = PLACE_X + Pw + 1 = 25, total_h = PF_PLACE_Y + Ph + 1 = 36 — well inside the 762×1176 WSE-3 fabric.
