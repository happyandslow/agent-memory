How the A/B/C timings leave the chip — implementation map

Two independent reporter PEs each read their own low-32 TSC deltas, pack a 4-u32 burst, and emit it off-chip through a 1-PE forwarder (kv_prof_mux.csl) to a host D2H stream. There are two separate host streams today — the suspected cause of the device-side Empty ingress service url hang. Every box is annotated with its source location so it can be inspected directly.
