# nc_service Memory

Curated agent/human memory for this project.

## Files

- `project.md` — stable facts, repo paths, machines, commands, conventions.
- `context.md` — compact startup context for fresh sessions.
- `topics/` — topic-specific packets.
- `transcripts/` — indexes/events/pointers; not raw transcript dumps by default.
- `inbox/` — temporary capture before curation.
- `agents/` — host/agent-specific setup notes.

## Policy

- Keep context compact and high-signal.
- Prefer durable facts and decisions over blow-by-blow logs.
- Do not commit secrets or full raw transcript stores.
- Curate `inbox/` periodically.
